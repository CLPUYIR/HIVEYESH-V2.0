# 🐝 HYVEYESH v2.0
### The Sovereign Swarm Architecture for Autonomous Distributed Edge Intelligence

Hyveyesh v2.0 is a maximalist, distributed AI orchestrator designed to transform isolated Windows LANs into a unified, sovereign edge computing platform. It shards the world's largest open-weight models across heterogeneous hardware, prioritizing intelligence over raw speed.

## 🚀 Quick Start
1.  **Stage 1 Initialization:**
    ```powershell
    cd management_plane
    .\Hyveyesh-Installer.ps1
    ```
2.  **Kernel Optimization:**
    ```powershell
    .\network_tuner.ps1
    ```
3.  **Start the Sovereign Gateway:**
    ```python
    python sovereign_layer/whatsapp_bridge.py
    ```

## 🏗️ Architecture
Hyveyesh v2.0 is built on three hierarchical layers:
- **Sovereign Layer:** Strategic planning, multi-agent routing (God Agent), and Shared Soul memory management.
- **Data Plane:** High-speed binary execution, hardware profiling, and sub-cluster gateway management.
- **Management Plane:** Low-level environment provisioning, multi-threaded weight distribution, and network kernel tuning.

See `documentation/BLUEPRINT.md` for the full technical specification.

## 📜 Identity & Philosophy
- **Maximalist:** If it fits in the collective RAM, it runs. No model is too big.
- **Sovereign:** Headless, modular, and fully decoupled from front-end messaging hooks.
- **Self-Healing:** Real-time node monitoring and transactional rollbacks.
